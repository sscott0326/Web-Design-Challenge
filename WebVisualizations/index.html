Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. Data was pulled from the OpenWeatherMap API to assemble a random dataset consisting of over 500 cities. Using Python, a series of scatterplots was produced to showcacse the relationship between a citie's latitude against 4 variables: temperature, humidity, cloudiness, and wind speed.
